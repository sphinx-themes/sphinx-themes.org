PyData Sample#

This is a demo site for PyData, generated as part of Sphinx Themes Gallery.

Important

This sample documentation was generated on Apr 28, 2025, and is rebuilt weekly.

Quickstart#

  1. Install this theme:

    $ pip install pydata-sphinx-theme

  2. Set the following in your existing Sphinx documentation’s conf.py file:

    html_theme = 'pydata_sphinx_theme'

  3. Build your Sphinx documentation with the new theme! [1]

Documentation#

You can find this theme’s documentation at https://pydata-sphinx-theme.readthedocs.io/en/latest/.
